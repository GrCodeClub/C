<arpa/...>

  1. <arpa/inet.h>: Αυτή η επικεφαλίδα περιέχει δηλώσεις και συναρτήσεις που σχετίζονται με τη μετατροπή και τη διαχείριση διευθύνσεων IP (Internet Protocol) και πόρτων.
  2. <arpa/tftp.h>: Αυτή η επικεφαλίδα περιλαμβάνει δηλώσεις που αφορούν το TFTP (Trivial File Transfer Protocol), που χρησιμοποιείται για τη μεταφορά αρχείων μέσω δικτύου.
  3. <arpa/nameser.h> και <arpa/nameser_compat.h>: Αυτές οι επικεφαλίδες περιλαμβάνουν δηλώσεις που αφορούν το πρωτόκολλο DNS (Domain Name System), που χρησιμοποιείται για τη μετάφραση ονομάτων κειμένου σε διευθύνσεις IP και αντίστροφα.
  4. <arpa/inetd.h>: Αυτή η επικεφαλίδα περιέχει δηλώσεις και συναρτήσεις που σχετίζονται με τον διαχειριστή υπηρεσιών (inetd), που είναι υπεύθυνος για την εκκίνηση δικτυακών υπηρεσιών όταν αυτές ζητούνται από το δίκτυο.
  5. <arpa/ftp.h>: Αυτή η επικεφαλίδα περιλαμβάνει δηλώσεις που αφορούν το FTP (File Transfer Protocol), ένα πρωτόκολλο που χρησιμοποιείται για τη μεταφορά αρχείων μεταξύ δικτυακών συστημάτων.
